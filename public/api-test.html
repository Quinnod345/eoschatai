API Endpoints

POST /api/v1/chat - Chat completions (OpenAI-compatible)
POST /api/v1/conversations - Create new conversation
GET /api/v1/conversations - List conversations
GET /api/v1/conversations/:id - Get conversation with messages
DELETE /api/v1/conversations/:id - Delete conversation
POST /api/v1/conversations/:id/messages - Send message to conversation
POST /api/v1/embeddings - Generate text embeddings
POST /api/v1/documents/analyze - Analyze document and answer questions
GET /api/v1/models - List available models
GET /api/v1/usage - Get API key usage statistics
