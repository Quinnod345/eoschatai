# Demo Videos for Upgrade Modal

This directory should contain demo videos for premium features:

- `export-demo.mp4` - Shows exporting chats/documents to PDF/DOCX
- `calendar-demo.mp4` - Shows Google Calendar integration
- `recordings-demo.mp4` - Shows voice recording and transcription
- `research-demo.mp4` - Shows deep research feature with web lookups

Each video should be:
- 10-15 seconds long
- Show the feature in action
- Highlight the value proposition
- Be optimized for web (compressed)

Alternative GIFs can be placed in `/public/gifs/` as fallbacks.
