## Usage

1. Add your EOS knowledge documents to the appropriate namespace directories
2. Run the upload script: `npm run upload-knowledge`
3. Documents will be processed and added to the system RAG database

## Document Format

- Use Markdown format (.md files)
- Include clear headings and structure
- Focus on actionable EOS guidance
- Include examples and best practices
