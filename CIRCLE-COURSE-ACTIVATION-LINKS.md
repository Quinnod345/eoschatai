## 🔧 Embedding in Circle.so

To add these links directly in your Circle.so courses:

1. Go to your Circle course settings
2. Add a button or link in the course description
3. Use the activation URL for that course
4. When users click, they'll be auto-redirected to your app

### Example Button HTML for Circle

```html
<a href="https://eoschat.ai/api/circle/activate-course?courseId=782928&audience=implementer" 
   style="display:inline-block;padding:12px 24px;background:#FF6B35;color:white;border-radius:8px;text-decoration:none;font-weight:600;">
   🤖 Activate AI Course Assistant
</a>
```

## ⚡ What Happens When User Clicks

1. **Check Authentication** - Redirect to login if needed
2. **Check Existing** - If user already has this course, return it
3. **Query RAG** - Get course content from Upstash (~1 sec)
4. **GPT-4.1** - Generate personalized instructions (~3 sec)
5. **Create Persona** - Save user's course assistant (~1 sec)
6. **Success** - Return persona data

**Total Time: ~5 seconds**
